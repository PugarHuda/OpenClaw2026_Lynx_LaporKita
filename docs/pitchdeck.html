Solution

Satu agent otonom: lapor → klasifikasi → routing → tracking → reward

Warga cukup kirim foto. AI multi-agent menyelesaikan sisanya — sampai masalah beres dan warga dapat reward.
1 · Lapor
Warga kirim foto + deskripsi via WhatsApp / web
2 · Proses
Agent klasifikasi (Claude vision), rute ke instansi, submit ke Lapor.go.id
3 · Lacak
Agent poll status otonom, eskalasi bila mangkrak
4 · Reward
Masalah beres → warga dapat Rasain Points (RSN) on-chain
Civic Credit warga tukar RSN untuk bayar retribusi pemerintah: RSN di-burn on-chain (Solana) → sisa tagihan dibayar via DOKU QRIS Berkontribusi melapor = membantu bayar kewajiban sipil sendiri.

AI Agent Workflow & Architecture

7 agent spesialis + autonomous loop

Citizen ─▶ Intake ─▶ Classifier (Claude vision) ─▶ Geolocator │ ▼ Submitter ─▶ Lapor.go.id │ Reward ◀── Verifier ◀── Tracker (cron loop · zero human trigger) │ ├──▶ Solana SPL : mint / burn Rasain Points (proof of impact) └──▶ DOKU MCP : QRIS Civic Credit redemption
Autonomy
  • Event loop: laporan masuk memicu pipeline klasifikasi→routing→submit
  • Cron loop: tracker jalan tiap 20 dtk tanpa trigger manusia
  • Decision branch: invalid→tolak, low-confidence→minta foto ulang, critical→fast-track
Reasoning Trace
  • Tiap langkah menulis jejak pikir natural-language
  • Dashboard streaming reasoning agent secara live
  • Tool call dinamis: vision, payment, blockchain
